---
title: nslookup
description: Query Internet name servers interactively.
---

O utilitário de texto `nslookup`, nativo em todos os sistemas operacionais, é utilizados para consulta de zona reversa de DNS.

It querires DNS records for a given domain name.

It allows you to view all the DNS records for a website.

Exemplificando, ao realizar a pesquisa da zona reversa da plataforma Gupy, utiliza-se o comando da seguinte forma:

```bash
nslookup www.gupy.io
```
