---
title: Dig
description: DNS lookup utility.
---

O utilitário de texto `dig` extrai informações de DNS.

Exemplo de uso:
```bash
dig navinet.com.br -t ns
```

```bash
; <<>> DiG 9.18.24-0ubuntu5-Ubuntu <<>> navinet.com.br -t ns
;; global options: +cmd
;; Got answer:
;; ->>HEADER<<- opcode: QUERY, status: NOERROR, id: 14040
;; flags: qr rd ra; QUERY: 1, ANSWER: 2, AUTHORITY: 0, ADDITIONAL: 1

;; OPT PSEUDOSECTION:
; EDNS: version: 0, flags:; udp: 65494
;; QUESTION SECTION:
;navinet.com.br.			IN	NS

;; ANSWER SECTION:
navinet.com.br.		3600	IN	NS	ns2.navinet.com.br.
navinet.com.br.		3600	IN	NS	ns1.navinet.com.br.

;; Query time: 994 msec
;; SERVER: 127.0.0.53#53(127.0.0.53) (UDP)
;; WHEN: Thu Nov 20 22:41:09 -03 2025
;; MSG SIZE  rcvd: 79

```

Para localização de servidores de **e-mail**:
```bash
dig navinet.com.br -t ns
```
