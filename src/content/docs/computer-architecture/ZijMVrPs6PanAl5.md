## 1. What's A Architecture?

It is a concept that describes how a component should work with a digital machine, e.g., a computer.

## 2. Von Neumann Architecture

Design created by American-Hungarian mathematician John von Neumann in 1945.

### 2.1. Components:

- Central Processing Unit

- Memory Unit

- Input/Output (I/O)

- Bus/Pathway

### 2.2. Concepts:

- Stored-Program Concept

- Sequential Execution

- Von Neumann Bottleneck
