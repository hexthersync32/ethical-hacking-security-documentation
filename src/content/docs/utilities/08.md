---
title: host
description: Send ICMP ECHO_REQUEST to network hosts.
---

## 1. Definition

A simple utility for performing forward and reverse DNS lookups.

## 2. Example Usage

###  2.1. Main Usage:

```bash
host linkedin.com
```

Result:

```bash
    linkedin.com has address 150.171.22.12
    linkedin.com has IPv6 address 2620:1ec:50::12
    linkedin.com mail is handled by 10 mail-c.linkedin.com.
    linkedin.com mail is handled by 20 mail.linkedin.com.
    linkedin.com mail is handled by 10 mail-a.linkedin.com.
    linkedin.com mail is handled by 10 mail-d.linkedin.com.
```

### 2.2. Reverse Lookup for IP:

```bash
host 8.8.8.8
```

Result:

```bash
    8.8.8.8.in-addr.arpa domain name pointer dns.google.
```

### 2.2. Name Server Records:
```bash
host -t NS wikipedia.org
```

Result:

```bash
    wikipedia.org name server ns2.wikimedia.org.
    wikipedia.org name server ns1.wikimedia.org.
    wikipedia.org name server ns0.wikimedia.org.
```
