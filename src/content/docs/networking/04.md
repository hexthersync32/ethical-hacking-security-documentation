## 1. Definition

Human-friendly, unique address used to access website through a browser.

It acts as a memorable alias for a complex numerial IP address.

For example: `google.com`.

## 2. Structure

Uma string de caracteres alfanuméricos, um ponto e mais uma informação logo após este ponto não é a única composição de um nome de domínio.

Um nome de domínio é hierárquico e lido da **direita para a esquerda**. Portanto, leve em consideração o seguinte exemplo:

> hireme.now.enterprise.com.ca

Dividindo o nome de domínio acima:

```txt
    ca → TLD
    com → Domínio de segundo nível
    enterprise → Domínio registrado
    now → Subdomain
    hireme → Subdomain
```

## 3. TLD

O Top-Level Domain (TLD) é a parte final do nome de domínio, estando localizado após o último ponto, indicando uma categoria ou país de origem.

## 6. ASN

An Autonomous System is a large network or colletion of IP networks under a singles administrative domain (like an ISP, university, or corporation) that sahres a unified routing policy.

This network manage traffic flow across the internet, enduring data packets efficiently find the correct route.

Also, every AS is assigned a unique Autonomous System Number (`ASN`) (16-bit or 32-bit number) for identification in the BGP routing table.

Quando as empresas compram um AS, elas conseguem fazer um roteamento com redes de outras empresas.

Essas empresam possuem uma autoridade administrativa sobre uma faixa de IP.

Caso seja consultado um AS, é possível identificar quais faixas de IP pertencentes à empresa estão disponíveis.
